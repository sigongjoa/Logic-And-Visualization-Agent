Project: ATLAS - 커리큘럼 지식 그래프 명세서 (V1)
1. 개요
본 문서는 Project: ATLAS의 '커리큘럼 축'을 정의하기 위한 V1 지식 그래프의 구성을 명세합니다. 이 명세는 Curriculums, Concepts_Library, Concept_Relations 테이블을 채우는 기준이 됩니다. [cite: atlas_database_spec.md]

*   Curriculums (커리큘럼): 학년별/과목별 대분류입니다.
*   Concepts_Library (개념 노드): AI가 분석하고 코치가 티칭하는 지식의 최소 단위입니다.
*   Concept_Relations (개념 엣지): "선수과목" 또는 "심화과정"을 정의하는 연결고리입니다.

2. Curriculum: M-ALL (중등 수학: 공통 기초)
4축 모델의 '인지 기저(axis1_*)'를 형성하는 가장 근본적인 뿌리입니다. 이 단계의 노드(개념)가 부실하면 고등 과정 전체가 무너집니다.

| 핵심 단원 (대분류) | 핵심 개념 (Concepts_Library 노드) | 주요 연결고리 (Concept_Relations 엣지) |
| :--- | :--- | :--- |
| 문자와 식 | • 소인수분해 • 정수와 유리수 • 일차방정식의 풀이 • (핵심) 다항식의 연산 (덧셈, 뺄셈, 곱셈) • (핵심) 곱셈 공식 및 인수분해 | • C_일차방정식 → REQUIRES → C_정수와유리수 • C_인수분해 → REQUIRES → C_다항식의연산 • (고등) C_다항식(고) → REQUIRES → C_인수분해(중) |
| 함수 | • 좌표평면과 그래프 • 정비례와 반비례 • (핵심) 일차함수와 그래프 (기울기, y절편) • (핵심) 이차함수와 그래프 (표준형, 일반형) | • C_일차함수 → REQUIRES → C_일차방정식 • C_이차함수 → REQUIRES → C_인수분해 • (고등) C_함수(고) → REQUIRES → C_이차함수(중) |
| 기하 | • 기본 도형 (점, 선, 면, 각) • 삼각형/사각형의 성질 • (핵심) 도형의 닮음 • (핵심) 피타고라스의 정리 • (핵심) 삼각비 (sin, cos, tan) | • C_피타고라스 → REQUIRES → C_삼각형의성질 • C_삼각비 → REQUIRES → C_피타고라스 • (고등) C_삼각함수(수1) → REQUIRES → C_삼각비(중) |
| 확률과 통계 | • 경우의 수 (합/곱의 법칙) • 확률의 기본 성질 | • (고등) C_순열과조합(고) → REQUIRES → C_경우의수(중) |

3. Curriculum: H-COMMON (고등 수학 (상), (하))
중등 기초를 '통합'하고 '심화'시키는 가장 중요한 허리입니다. 4축 모델의 '인지 기저' 3가지(대수, 기하, 해석)가 본격적으로 결합되는 시기입니다.

| 핵심 단원 (대분류) | 핵심 개념 (Concepts_Library 노드) | 주요 연결고리 (Concept_Relations 엣지) |
| :--- | :--- | :--- |
| 다항식(고등수학 상) | • (핵심) 항등식과 나머지 정리 • (핵심) 인수분해 (고차식) • 복소수 (i의 정의) | • C_나머지정리 → REQUIRES → C_다항식의연산(중) • C_복소수 → REQUIRES → C_제곱근(중) |
| 방정식과 부등식(고등수학 상) | • (핵심) 이차방정식 (근의 공식, 판별식) • (핵심) 이차함수와 이차방정식의 관계 • 여러 가지 방정식 (3/4차, 연립) | • [매우 중요] C_이차함수와관계 → LINKS → C_이차함수(중) • [매우 중요] C_이차함수와관계 → LINKS → C_이차방정식 • (해석학의 기초, axis1_ana) |
| 도형의 방정식(고등수학 상) | • 평면좌표 (두 점 사이의 거리) • 직선의 방정식 • (핵심) 원의 방정식 • 도형의 이동 (평행/대칭이동) | • [매우 중요] C_원의방정식 → LINKS → C_피타고라스(중) • [매우 중요] C_원의방정식 → LINKS → C_이차함수(중) • (기하(axis1_geo)와 대수(axis1_alg)의 연결) |
| 집합과 명제(고등수학 하) | • 집합의 연산 • 명제 (필요/충분조건) | • (수학의 '언어'. 모든 상위 개념의 논리적 기반) |
| 함수(고등수학 하) | • (핵심) 함수 (일대일, 항등, 상수) • (핵심) 합성함수와 역함수 • 유리함수와 무리함수 | • C_합성/역함수 → REQUIRES → C_함수 • (미적분의 핵심 도구, axis1_ana의 핵심) |
| 경우의 수(고등수학 하) | • (핵심) 순열 (nPr) • (핵심) 조합 (nCr) | • C_순열과조합 → REQUIRES → C_경우의수(중) • (확률과 통계의 선수과목) |

4. Curriculum: H-SELECT (고등 수학: 선택 과목)

5. 결론 및 활용 방안
이 명세서는 코치님이 Project: ATLAS의 '지식 그래프'를 구축하기 위한 V1 청사진입니다.

*   **즉시 활용:** Curriculums와 Concepts_Library 테이블에 이 데이터(개념 노드)를 INSERT하여 V1 뼈대를 시작할 수 있습니다.
*   **V1 핵심 기능:** Meta-RAG 엔진은 학생이 문제를 냈을 때, 이 Concepts_Library를 검색하여 가장 관련성 높은 `concept_id`를 찾아내고, `manim_data_path`에 연결된 유튜브 영상(V1)을 학생에게 제공합니다.
*   **V2 확장:** Concept_Relations 테이블(엣지)을 구축하면, AI가 "이 학생은 C-005가 약하니, 선수과목인 C-002를 먼저 복습시켜야 합니다"와 같은 **'학습 경로 추천'**까지 가능하게 됩니다.
